配置邮箱设置

### 验证码

```
auto_verify=True # 自动识别验证码
```

自动识别验证码使用的是[图鉴平台](http://ttshitu.com/)，可以自行申请。

```
auto_verify=False # 手动识别验证码
```

图片自动保存在当前文件夹/captcha，查看后在控制台直接输入即可。

### 自动模式

```
auto_mode=True # 自动选课
```

请手动更改name_list

```
auto_mode=False # 手动选课
```

手动输入课程序号即可。
